#Olá! Neste documento irei explicar o passo a passo para a criação do script para execução do programa "calc_basic.py" através do script "Calculadora Básica - Python.sh"
# 1 - Foi criado o arquivo "Calculadora Básica - Python.sh" utilizando o comando:
# -> $nano "Calculadora Básica - Python.sh"
# 2 - Depois foi adicionada a permissão de execução com o comando:
# -> $nano +x "Calculadora Básica - Python.sh"
# 3 - Depois foi dado ao usuário (root) a permissão de escrita com o comando:
# -> $chmod +w "Calculadora Básica - Python.sh"
# 4 - Por ultimo foi dado aos outros usuários a permissão de leitura com os seguintes comandos:
# -> $chmod o+r "Calculadora Básica - Python.sh" && chmod g+r "Calculadora Básica - Python.sh"
